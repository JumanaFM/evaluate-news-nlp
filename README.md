# Instructions

Creat .env file on the project root dirctory, then add the api key for **MeaningCloud** as follow

```
API_KEY=**************************
```

Run the command

```
npm install
```

---

## For Dev

Run the following command in one termanal

```
npm run build-dev
```

and in another termanal run the command

```
npm start
```

## For Prod

Run the commands

```
npm run build-prod
npm start
```
